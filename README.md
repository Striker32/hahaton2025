# README

Для работы необходимо:
- Склонировать репозиторий
- Для установки зависимостей прописать в консоли, находясь в корневой директории проекта:
```
pip install -r requirements.txt
```

- Запустить файл web_server.py
- Перейти в браузер по ссылке http://localhost:5000 (по желанию ссылку можно поменять, перейдя в конец файла web_server.py)
